# Portfolio_Website
A modern, responsive portfolio website showcasing my projects, skills, and achievements, designed to provide an engaging user experience with clean layouts, smooth navigation, and mobile-friendly design.

---

## Features

- **Responsive Design** – Optimized for all screen sizes (mobile, tablet, laptop, desktop)
- **Contact Form with Google Sheets Integration** – Automatically stores form submissions to a connected Google Sheet
- **Modern UI** – Clean and professional look with smooth navigation
- **Fast & Lightweight** – No heavy frameworks, just pure HTML, CSS, and JavaScript
- **Cross-Browser Compatible** – Works across major browsers

---

##  Technologies Used

- **HTML5** – Structure & content
- **CSS3** – Styling & responsiveness
- **JavaScript** – Interactivity
- **Google Sheets** – Backend for contact form submissions
